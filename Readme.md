# sift

  Fast String Distance (SIFT) Algorithm.

## Installation

    $ component install timoxley/sift

## Demo

[Demo](http://timoxley.github.com/sift/examples/spellcheck/)

## API

### sift(string, string)

  Return 'distance' between two strings.
